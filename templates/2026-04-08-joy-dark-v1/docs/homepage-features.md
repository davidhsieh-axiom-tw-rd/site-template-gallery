## 10. 「更多」選單頁面

點擊底部 Tab Bar 的「更多」tab 後展開全屏覆蓋頁面。

### 結構
- Header：返回箭頭 + 「更多」標題
- 5 個功能區塊，每區一個圓角卡片容器：

| 區塊 | 功能項目 |
|------|---------|
| 首页常用功能 | 分享赚钱, 返水, VIP, 客服（可编辑） |
| 优惠中心 | 活动, 任务, 盲盒抽奖, 幸运转盘, 待领取, 领取记录, 新砍一刀 |
| 资金管理 | 存款, 提现, 客服代充, 账户明细, 投注记录, 个人报表, 提现管理, 立即分享, 投注任务 |
| 支持中心 | 语言, APP下载, 有奖反馈, 朋友圈 |
| 账户安全 | 个人资料, 安全中心, 自我禁止 |

### Flutter 對應
- 整個「更多」頁面 = 獨立 Route/Screen（`MoreMenuScreen`）
- 每個區塊 = `Card` + `Wrap` 或 `GridView`
- 功能項目 = `InkWell` + `Column`（icon + text）
- Header = `AppBar` with back button
