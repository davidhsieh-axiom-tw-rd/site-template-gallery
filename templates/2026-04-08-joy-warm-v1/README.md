# Joy Warm V1 — 暖橘棕色主題

開元棋牌 (joy.star-link-rel.cc) 首頁版型復刻。

## 快速開始
```bash
cd templates/2026-04-08-joy-warm-v1
python3 -m http.server 8080
# 瀏覽 http://localhost:8080
# 凍結模式 http://localhost:8080?freeze=true
```

## 版型特色
- 暖橘棕色調（#FF6F00 主色 + #783D21 背景）
- 山脈森林 header 背景
- 3 列遊戲卡片 grid
- 側邊欄選單（电子游戏/娱乐城/优惠中心/简体中文/支持中心）
- 中獎輪播 + 跑馬燈
- 底部 5 Tab Bar

## 目錄
- `index.html` — 版型主體（CSS/JS inline）
- `metadata.json` — 版型元資料
- `screenshot.png` — 原始站截圖
- `assets/` — 圖片資源本地備份
- `docs/` — Flutter 轉換文件

## 瀏覽器支援
Chrome 90+, Safari 15+, Firefox 90+
